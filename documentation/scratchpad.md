## Agent Flow System
The system uses a 6-agent workflow for feature development:
1. **Quick Assessment** (`design-0-quick.md`) - Triage for simple changes
2. **Planning** (`design-1-planning.md`) - Initial analysis and requirements capture
3. **Review** (`design-2-review.md`) - Validation and quality checks  
4. **Discovery** (`design-3-discovery.md`) - Technical research and codebase analysis
5. **Execution** (`design-4-execution.md`) - Implementation
6. **Visual Verification** (`design-5-visual-verification.md`) - Screenshot testing across breakpoints
7. **Completion** (`design-6-complete.md`) - Final documentation and learning
